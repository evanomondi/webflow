BimaExchange| Web Flow

  • Step 1: User accesses The Landing page

    Page: Landing Page

    When User enters the landing Page they Fill in the details

  • Step 2: Congratulations Page

    Page: Congratulations Page

    User Gets a congratulatory notification With a coupon code. An email is sent there after

  • Step 3: Insurance type information

    Page: Insurance Profile

    User fill in information that is used by the website to provide insurance quotes

  • Step 4: Choosing the principal

    Page: Primary Section

    User chooses if he/she is the principal. If not H/She provides the principal's information

  • Step 5: Quotes Page

    Page: Quotes

    User is able to view and select His/Her quotes based on information they provided

  • Step 5: Secondary Information

    Page: Secondary Information

    User will fill in secondary information

  • Step 6: Creating a dependant

    Page: Secondary Information

    UUser adds a dependant on this section. Keying in the dependants details

  • Step 7: Delivery Information

    Page: Delivery Information

    User Specifies where he/she wants the Insurance documentation deliverd. They have an option of picking up or having them delivered

  • Step 8: Coupon

    Page: Coupon

    User has an opton of using a coupon( A Discounted code used to give a discount on the system)

  • Step 9: Order Overview/Checkout

    Page: Overview/Checkout

    User is able to view the products he/she has chosen with their prices and can choose to add more options i.e optional benfits e.g outpatient cover

  • Step 10: Disclaimer

    Page: Overview/Checkout

    User Agrees to a disclaimer which states that he/she should upload the necessary documents.

  • Step 11: Payment

    User Choses how to pay for athe order and pays for it

  • Step 12: Tertiary information

    User ills out tertiary information. i.e ID number or passport number

  • Step 13: Uploading of Documents

    User uploads the necessary documents required by the underwriter for health insurance
